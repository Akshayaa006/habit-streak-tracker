# 🔥 Habit Streak Tracker

![CI/CD](https://github.com/Akshayaa006/habit-streak-tracker/actions/workflows/docker-build.yml/badge.svg)

A simple Habit Tracker application built using HTML, CSS, and JavaScript.

## Features

- Track daily habits
- Monitor streak counts
- Clean and responsive UI
- Dockerized application
- GitHub Actions CI/CD pipeline
- SonarQube code quality analysis

## Technologies Used

- HTML
- CSS
- JavaScript
- Docker
- GitHub Actions
- SonarQube

## Run Locally

```bash
docker build -t habit-tracker .
docker run -d -p 8080:80 habit-tracker
